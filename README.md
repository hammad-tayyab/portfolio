# Hammad Tayyab — Portfolio

A minimalist, rugged personal portfolio website showcasing my work, experience, and projects.

Access the live site at: **[hammad-tayyab.vercel.app](https://hammad-tayyab.vercel.app/)**

## Features

- **Minimalist & Rugged Aesthetic**: Clean, high-contrast layout with an off-white background and charcoal typography.
- **Dark Mode Toggle**: Fluid theme switching saved locally.
- **Wandering Ladybug**: A custom SVG ladybug that strolls across the page leaving a light canvas-fading trail.
- **Custom Cursor**: Interactive red accent cursor that adapts to hover triggers.
- **Typewriter Effect**: Dynamic hero subtitles detailing my latest role and skills.
- **Single-File Architecture**: Clean vanilla HTML, CSS, and JavaScript with zero external frameworks for lightning-fast loads.

## Skills & Stack

- **Languages**: Python, C++, C, SQL, HTML, CSS
- **AI / ML**: LLMs, Agentic AI, Agents, Supervised Learning, NLP, Data Analysis, Pandas, NumPy, Seaborn
- **Web**: React, Tailwind CSS, Supabase
- **Tools**: Git, GitHub, Linux, API Integration, Printify

## Local Development

To run the portfolio locally, serve it using any static file server (e.g., Python's built-in HTTP server):

```bash
# Serve locally
python -m http.server 8080
```

Then open `http://localhost:8080` in your web browser.
